# Northwind Power BI Project

## Description
This project analyzes the Northwind database using Power BI.
The data source is a PostgreSQL database imported from the Northwind SQL script.

## Tools Used
- PostgreSQL
- DataGrip
- Power BI Desktop
- GitHub

## Reports Included
1. Total Sales by Product Category
2. Total Sales by Customer Country
3. Top 10 Customers by Total Sales

## Measure Used
Total Sales is calculated using the formula:
Unit Price * Quantity.

## Notes
The PostgreSQL database is hosted locally.
The Power BI file reconnects to the database when opened.
